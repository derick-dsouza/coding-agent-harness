# Spec Templates

This directory contains templates for creating application specifications that work optimally with AI coding agents.

## Files

- **`app_spec_template.xml`** - Comprehensive XML template for defining application requirements
- **`TEMPLATE_GUIDE.md`** - Guide for using the template effectively

## Quick Start

```bash
# Copy template to your project
autospec

# Or specify directory
autospec /path/to/project
```

This creates `app_spec.txt` in the target directory with the full XML template.

## Why XML?

XML provides:
- **Hierarchical structure** for organizing complex requirements
- **Clear boundaries** between sections
- **Parser-friendly** format for AI agents
- **Validation** capabilities
- **Natural language** content within structured tags
